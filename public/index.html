SkinCraft API

Access Minecraft player skins, capes, and profile information

API Endpoints

The SkinCraft API provides the following endpoints for accessing Minecraft player data:

GET /api/user/:identifier/profile

Get complete profile information for a player by username or UUID

Parameters:

  • identifier - Minecraft username or UUID

Response: Player profile including name history, skins, and capes

GET /api/user/:identifier/capes

Get all capes owned by a player

Parameters:

  • identifier - Minecraft username or UUID

Response: List of capes with details and usage count

GET /api/user/:identifier/skins

Get all skins owned by a player

Parameters:

  • identifier - Minecraft username or UUID

Response: List of skins with tags and usage count

GET /api/capes

Get a list of all known Minecraft capes

Response: Complete list of Minecraft capes with descriptions

GET /api/names/:length

Get soon-to-be-available Minecraft names by character length

Parameters:

  • length - Number of characters (3-16)

Response: List of upcoming available names with availability dates and OG status

GET /api/name/:username

Check if a specific Minecraft username is available or when it will become available

Parameters:

  • username - The Minecraft username to check

Response: Information about username availability and release date (if applicable)

GET /api/names

Get soon-to-be-available Minecraft names (first 10)

Response: List of upcoming available names with availability dates and OG status

GET /api/skins/latest

Get the latest Minecraft skins uploaded by players

Response: List of the 6 most recent skins with player names and skin IDs

GET /api/skins/random

Get random Minecraft skins

Response: List of randomly selected Minecraft skins with their IDs

GET /api/skins/daily

Get daily trending Minecraft skins

Response: List of trending skins with player names (if available), skin IDs, and popularity counts

GET /api/skins/weekly

Get weekly trending Minecraft skins

Response: List of trending skins with player names (if available), skin IDs, and popularity counts

GET /api/skins/monthly

Get monthly trending Minecraft skins

Response: List of trending skins with player names (if available), skin IDs, and popularity counts

Usage Examples

Fetch a player profile


// Using fetch
fetch('https://skincraft-api.vercel.app/api/user/Notch/profile')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

// Using axios
axios.get('https://skincraft-api.vercel.app/api/user/Notch/profile')
  .then(response => console.log(response.data))
  .catch(error => console.error('Error:', error));
